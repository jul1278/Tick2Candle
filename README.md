# Tick2Candle
Command line app to convert tick data to OHLC candles 


# Arguments
--t "path to tick file directory"__
--o "path to candle output directory"__
--ask col number__
--bid col number__
--time col number__
--ua use ask__
--ub use bid__
--um  use mid__
--ph period hours__
--ps period second__
